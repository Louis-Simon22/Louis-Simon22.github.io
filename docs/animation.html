In orderd to start an animation you need to add to your element:

START | ANIMATION TYPE | DURATION | DELAY (optional) | CUBIC BEZIER (optional)


Example: 

<h1 start animation=fadein ease=easeInOutQuad style="--duration: 1s; --delay: 1s;">Skelet.</h1>

 Animation var() style="--duration: 1s"

Define animation duration

 style="--delay: 1s"

Define animation delay [optional]

 style="--ease: ..."

Define your own rate of change over time.
i.e. Cubic Bezier [optional]

 If you do not specify the duration of the animations, it defaults to 1s. Delay is set to 0. Utility classes start/.animate

To start an animation you need either one or the other.

 .invisible/.hide

Setting element's opacity to 0, good for hiding elements when animation starts.

 .infinite/.loop

Loops the animation forever

 .reverse

Specifies whether an animation should play in reverse direction

 .alternate

The animation will be played as normal every odd time (1,3,5,etc..) and in reverse direction every even time (2,4,6...)

 .alternate-reverse

The animation will be played in reverse direction every odd time (1,3,5,etc..) and in a normal direction every even time (2,4,6,etc...)
