Dark mode

In case you're building a Dark website, you can set Skelet. automatically in dark mode, doing so you'll have everything set in darker shades instead of light.

To activate "Dark mode" you need to add one of the options below to <body> tag

 data-mode="dark"

This is one way of activating the Dark mode

 .dark-mode

This way two of activating the Dark mode

 #dark-mode

This way three of activating the Dark mode

Tooltips

To use Tooltips simply insert your text inside data-tooltip and use one of the data-tooltip-pos from the list below to position the tooltip.
  • data-tooltip-pos="up"
  • data-tooltip-pos="down"
  • data-tooltip-pos="left"
  • data-tooltip-pos="right"
