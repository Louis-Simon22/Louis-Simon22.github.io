Typeface

Skelet. makes use of The Inter typeface family.

Skelet. includes Inter through creator's provided CDN. If you need to host the font on your own server, head over here.

If you use skelet.css via CDN you need to include Inter in your <head> as well.

Paragraph

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

Horizontal rules

You can add one of the classes below to the <hr> tag to style it.

 .full Makes hr full-width .gradient Makes the hr line dim on the edges. style="--hrMargin: 5px" Control yourself the top & bottom margins [no need to include .space] <hr size="5"> You can define the thickness of hr style="--hrColor: blue" You can define the color of hr <span class="text-hr"> To create a hr with text inside

Fluid text

Here's an example of how to make text change while resizing the viewport.

.skelet

.heading

.title

 
:root { 
    --fontSize: calc(100vw / 80); 
    --hSkelet: 3.5vw;
    --hHeading: 3vw;
    --hTitle: 2.5vw;
}

.skelet { font-size: var(--hSkelet); }
.heading { font-size: var(--hHeading); letter-spacing: 0 }
.title { font-size: var(--hTitle); letter-spacing: 0 }

@media (max-width: 600px) { 
    :root {
        --fontSize: 1.3rem; 
        --hSkelet: 4rem;
        --hHeading: 3rem;
        --hTitle: 2.5rem; 
    } 
}

@media only screen and (min-width: 1599px) {
    :root { 
        --percentage: 15%; 
        --fontSize: 1.8rem; 
        --hSkelet: 5rem; 
        --hHeading: 4rem; 
        --hTitle: 3rem;
    }
}
